Os tokens válidos em JSON são:

{ - Abre um objeto JSON.
} - Fecha um objeto JSON.
[ - Abre um array JSON.
] - Fecha um array JSON.
, - Separa dois valores em um objeto ou array JSON.
: - Separa a chave e o valor em um objeto JSON.
string - Uma string JSON é uma sequência de caracteres delimitada por aspas duplas.
number - Um número JSON é um número decimal ou um número inteiro.
true - O valor booleano true em JSON.
false - O valor booleano false em JSON.
